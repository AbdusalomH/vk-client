**VKontakte App **

1. 100% Programatically
2. UIKit
3. URLSession
4. AutoLayout,
5. SDWebImage
6. Kingfisher,
7. MVC,
8. MVP,
9. MVVM
10. Swift package manage (no cocoapods)
