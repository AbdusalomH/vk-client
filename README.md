**VKontakte App **

It's a pet project with social media platform (russian social media). Added several pages with Friends, Communities, Communites wall, Videos.

1. 100% Programatically
2. UIKit
3. URLSession
4. AutoLayout,
5. SDWebImage
6. Kingfisher,
7. MVC,
8. MVP,
9. MVVM
10. Swift package manage (no cocoapods)
